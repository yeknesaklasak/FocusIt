# FocusIt

FocusIt is a Python-based AI timer designed to help users maintain focus while working. It uses computer vision to track presence and manage study sessions effectively.

## Features
- **Presence Tracking:** Uses MediaPipe and OpenCV to monitor if the user is focused at their desk.
- **Smart Timer:** Automatically manages work and break intervals.
- **Focus Alerts:** Provides notifications to keep the user on track.

## Requirements
To run this project, you need to have Python installed along with the following libraries:
* OpenCV
* MediaPipe

You can install them using:
pip install opencv-python mediapipe
